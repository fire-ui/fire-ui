Box

Start using Fire UI box by simply add box class into your element. Fire UI also provides box-shadow and box-title class for your box element.

Example

Hello World

<div class="box box-shadow">
    <h1 class="box-title">Hello World</h1> 
</div>

Combination

You can combine box with other classes too! For example:

Hello

World

<div class="row">
    <div class="col-ex-6">
        <div class="box theme-reverse">
            <p class="box-title">Hello</p>
        </div>
    </div>
    <div class="col-ex-6">
        <div class="box theme-adjust">
            <p class="box-title">World</p>
        </div>
    </div>
</div>
